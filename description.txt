Задание

Закончить веб-приложение, которое отображает состояние версий ядра и расширений TYPO3, собранной с нескольких рабочих серверов.
Незаконченное приложение (на самом деле специально "обрезанное") находится в файле "source.zip", данные статистики - в файле "data.zip".
Как должно выглядеть законченное приложение - изображено на скриншоте "result.png".

Приложение не использует базу данных и читает данные напрямую из файлов статистики (статистика собрана за период, отображать надо последнее состояние).
Файлы статистики содержат данные по используемым расширениям и их версиям, используемым версиям основной системы TYPO3 и версиям PHP.
Файл "signature.txt" содержит реальные данные по версиям ядра и расширений для TYPO3 с уязвимыми версиями (на основании этих данных необходимо подсвечивать статусы в колонках TYPO3 и Extensions).
Формат файла несложный для понимания и содержит комментарии.

Результатом работы будет ваш законченный код и небольшое описание того, что вы сделали.
Для этого вам придётся самостоятельно развернуть незаконченное веб-приложение и доустановить зависимости, разобраться с форматами файлов и дописать код (как минимум в двух местах).
